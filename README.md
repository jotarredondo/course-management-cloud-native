# Course Management Cloud Native

Proyecto desarrollado para la Evaluación Final Transversal (EFT) de la asignatura **Desarrollo Cloud Native (CDY2204)**.


## Tecnologías

- Java 21
- Spring Boot 3.5
- RabbitMQ
- Docker
- AWS EC2
- AWS S3
- AWS API Gateway
- Azure Entra ID
- GitHub Actions

## Arquitectura

```text
Postman
    │
Azure Entra ID
    │
AWS API Gateway
    │
Producer (BFF)
    │
RabbitMQ
    │
Consumer
    │
H2 + AWS S3
```
